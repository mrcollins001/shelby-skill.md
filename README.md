# shelby-skill.md
building an AI agent that can autonomously interact with blockchain protocols and execute onchain actions for users. The goal is to create an intelligent assistant that can understand user intent and perform tasks such as deploying contracts, managing wallets, executing DeFi strategies, monitoring transactions, and interacting with smart contracts
The agent will combine large language models with onchain tooling so users can simply describe what they want to do and the agent translates that into secure blockchain operations. For example a user could ask the agent to bridge assets, swap tokens, deploy a contract, track wallet activity, or automate trading strategies.

I want to use Shelby’s infrastructure to give the agent reliable access to blockchain data and execution environments. This will help the AI operate with real time state, verify transactions, and safely interact with decentralized applications.

The long term goal is to build a universal onchain AI operator that lowers the barrier for interacting with Web3 while still giving developers powerful automation capabilities.
